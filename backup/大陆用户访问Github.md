# 大陆用户如何访问Github

## Github是什么？

>Github是一个**全球性开源平台**。在此平台上有适合不同人群的不同大量**优质开源资源及开源作品**——Github一直与**微软（Microsoft）**有着亲密的关系，是微软的**长期合作伙伴**，安全性双重保障，可以放心使用！Github主要面向**开发者群体**，所谓开源就是**开放源码**的意思。
